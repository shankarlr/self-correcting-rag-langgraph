Web Server Log Analysis

Description
This project analyzes real-world Apache web server logs from the University of Calgary to extract insights using Python.

Tools Used
- Python 3
- pandas
- re (regex)
- datetime
- tqdm
- Jupyter Notebook

Questions Answered
1. Total log count
2. Unique hosts
3. Unique filenames by date
4. Count of 404 errors
5. Top 404 filenames
6. Top 404 extensions
7. Bandwidth/day in July 1995
8. Hourly requests
9. Top requested files
10. HTTP status code distribution
